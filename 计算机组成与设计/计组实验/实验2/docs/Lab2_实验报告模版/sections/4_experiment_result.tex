\section{实验结果及分析}

\subsection{仿真图}
仿真波形如图\ref{fig:sim}所示。PC从0x00000000开始每个时钟周期递增4，指令存储器输出对应地址的32位指令，控制器输出的各路控制信号随指令类型正确变化。PC自增正确，各控制信号与预期译码结果一致。

\subsection{控制台输出}
仿真控制台输出如下，每行对应一个时钟周期的PC值、指令内容及各控制信号：

\begin{verbatim}
cnt=1 PC=00000004  instr=20020005  memtoreg=0 memwrite=0 pcsrc=0
      alusrc=1 regdst=0 regwrite=1 jump=0 branch=0 alucontrol=010
cnt=2 PC=00000008  instr=00a42820  memtoreg=0 memwrite=0 pcsrc=0
      alusrc=0 regdst=1 regwrite=1 jump=0 branch=0 alucontrol=010
cnt=3 PC=0000000c  instr=8c020050  memtoreg=1 memwrite=0 pcsrc=0
      alusrc=1 regdst=0 regwrite=1 jump=0 branch=0 alucontrol=010
cnt=4 PC=00000010  instr=ac020054  memtoreg=0 memwrite=1 pcsrc=0
      alusrc=1 regdst=0 regwrite=0 jump=0 branch=0 alucontrol=010
cnt=5 PC=00000014  instr=10a7000c  memtoreg=0 memwrite=0 pcsrc=0
      alusrc=0 regdst=0 regwrite=0 jump=0 branch=1 alucontrol=110
cnt=6 PC=00000018  instr=08000013  memtoreg=0 memwrite=0 pcsrc=0
      alusrc=0 regdst=0 regwrite=0 jump=1 branch=0 alucontrol=010
\end{verbatim}

各指令控制信号分析：
\begin{itemize}
    \item \textbf{addi}（instr=20020005）：alusrc=1（立即数），regwrite=1，alucontrol=010（加法），符合预期。
    \item \textbf{add}（instr=00a42820）：regdst=1（写rd），regwrite=1，alucontrol=010（加法），符合预期。
    \item \textbf{lw}（instr=8c020050）：memtoreg=1，alusrc=1，regwrite=1，alucontrol=010（地址计算），符合预期。
    \item \textbf{sw}（instr=ac020054）：memwrite=1，alusrc=1，regwrite=0，alucontrol=010，符合预期。
    \item \textbf{beq}（instr=10a7000c）：branch=1，alucontrol=110（减法判零），符合预期。
    \item \textbf{j}（instr=08000013）：jump=1，其余控制信号均为0，符合预期。
\end{itemize}
